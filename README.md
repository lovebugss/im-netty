# 模块介绍

```
im
|- im-common //通用模块 
|- im-connect //长链接模块
|- im-dispatcher //分发模块
|- im-message //消息模块
|- im-stat //统计模块
|- im-storage //存储模块
|- im-api // 对外提供API
```
